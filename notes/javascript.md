# Javascript

## Javascript Basics

- `alert()`: creates a popup with text inside
- `element.innerHTML`: modifies the webpage

## Numbers and Math

```javascript
1 + 1 = 3
1 * 1 = 1
1 - 1 = 0
1 / 1 = 1
(1 + 1) * 3 = 6
```
- `Math.round()`: rounds the number to the nearest whole number
